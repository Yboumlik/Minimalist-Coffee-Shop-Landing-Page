## Overview
This project showcases a "wild but minimalist" design aesthetic, using semantic HTML5, modern CSS (Grid/Flexbox, Variables), and vanilla JavaScript for interactivity.

## Features
- **Hero Video Background**: Immersive first impression.
- **Minimalist Typography**: Using `Syne` (Display) and `Space Mono` (Text).
- **Custom Cursor**: Interactive cursor with hover states.
- **Scroll Animations**: Elements reveal themselves as you scroll.
- **Responsive Design**: Fully responsive layout for all devices.

## Setup
1. Clone the repository.
2. Open `index.html` in your browser.
3. Enjoy the coffee experience.
